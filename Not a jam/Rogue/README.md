# Rogue

## How to launch

- only linux
- open folder in terminal
- press "make run" OR "./ROGUE"
- have fun)

## Controls

- WASD - move
- QE - change octaves
- ZX - change frequency
- C - regenerate world
- F - pay respect
- Escape - exit
